# Camera Controls

## Movement
| Key | Action |
|-----|--------|
| `W` | Move forward |
| `S` | Move backward |
| `A` | Move left |
| `D` | Move right |
| `E` | Move up |
| `Q` | Move down |
| `Left Shift` | Sprint (3x speed) |

## Look Around
| Input | Action |
|-------|--------|
| `Mouse` | Look around |
| `Scroll Wheel` | Zoom in/out |

## Cursor
| Key | Action |
|-----|--------|
| `Escape` | Toggle cursor lock/unlock |
